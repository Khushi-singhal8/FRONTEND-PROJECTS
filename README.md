# Frontend Projects — Khushi Singhal

A collection of frontend projects built while learning and practising HTML, CSS, JavaScript, and React. Each folder is a standalone project with its own code and assets.

An interactive portfolio with 3D visual effects built using HTML, CSS, and JavaScript.
- **Tech:** HTML · CSS · JavaScript
- **Highlights:** 3D transforms, smooth animations, responsive layout

- **Tech:** HTML · CSS · JavaScript
- **Highlights:** Scroll-triggered animations, parallax effects

### Animated Portfolio
A visually rich animated portfolio website.
- **Tech:** HTML · CSS · JavaScript

## How to Run Any Project Locally

```bash
# Clone the repo
git clone https://github.com/Khushi-singhal8/FRONTEND-PROJECTS.git

# Navigate to any project folder
cd FRONTEND-PROJECTS/3D_Interactive_Portfolio

# Open index.html in your browser
# For React projects:
npm install
npm start
```
